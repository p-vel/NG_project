Numerical Geometry Project: Convex Hulls
========================================

This folder contains our project on convex hulls.
More precisely, four algorithms have been implemented:
 * Graham Scan
 * Jarvis March
 * Chan Algorithm
 * Akl-Toussaint Heuristic.
